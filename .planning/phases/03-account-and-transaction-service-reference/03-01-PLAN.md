**get-account.mdx:**

Frontmatter: title "GetAccount", description "Fetch a Solana account's balance, owner, and on-chain data."

Opening: "Fetches the on-chain state of a Solana account by its Base58-encoded address. Returns the account's SOL balance in lamports, owner program, and raw account data."

## Request

ResponseField for `address`: type "Base58-encoded public key (string)", required. "The Solana account address to fetch."
ResponseField for `commitment_level`: type "CommitmentLevel (enum)", not required. "Optional. Controls which version of the ledger state to query. Defaults to CONFIRMED." Include link to /api-reference/shared-types.
Then include: `import CommitmentLevelNote from "/snippets/commitment-level-note.mdx"; <CommitmentLevelNote />`

## Response

ResponseField for `account`: type "Account (object)". "The account state." With Expandable containing:
- `address` (Base58-encoded public key (string)): "The account's address."
- `lamports` (uint64): "SOL balance in lamports. 1 SOL = 1,000,000,000 lamports."
- `owner` (Base58-encoded public key (string)): "Address of the program that owns this account."
- `executable` (bool): "True if this account contains a deployed program."
- `data` (string): "Account data encoded as a JSON string."
- `rent_epoch` (uint64): "Epoch at which this account will next owe rent."

## Code Examples

CodeGroup with Go/Rust/TypeScript (tab order: Go, Rust, TypeScript per D-10). Minimal examples — just the API call and key response usage (per D-11). Use `YOUR_ENDPOINT:50051` not localhost. Example addresses: use "7EcDhSYGxXyscszYEp35KHN8vvw3svAuLKTzXwCFLtV" for Go/Rust, same for TypeScript.

Go:
```go Go
resp, err := client.GetAccount(ctx, &account_v1.GetAccountRequest{
    Address: "7EcDhSYGxXyscszYEp35KHN8vvw3svAuLKTzXwCFLtV",
})
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Balance: %d lamports\n", resp.Account.Lamports)
fmt.Printf("Owner: %s\n", resp.Account.Owner)
```

Rust:
```rust Rust
let response = client.get_account(tonic::Request::new(GetAccountRequest {
    address: "7EcDhSYGxXyscszYEp35KHN8vvw3svAuLKTzXwCFLtV".to_string(),
    ..Default::default()
})).await?;
let account = response.into_inner().account.unwrap();
println!("Balance: {} lamports", account.lamports);
println!("Owner: {}", account.owner);
```

TypeScript:
```typescript TypeScript

req.setAddress("7EcDhSYGxXyscszYEp35KHN8vvw3svAuLKTzXwCFLtV");

  const account = response.getAccount();
  console.log("Balance:", account.getLamports(), "lamports");
  console.log("Owner:", account.getOwner());

```
