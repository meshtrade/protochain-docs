<ResponseField name="token_program" type="TokenProgram" required>
  Specifies which token program to use. Must be either `TOKEN_PROGRAM_LEGACY` (SPL Token) or `TOKEN_PROGRAM_2022` (Token Extensions). `TOKEN_PROGRAM_UNSPECIFIED` is invalid and returns an error. SPL Token and Token-2022 mints are incompatible — use the same program that was used to create the mint. See [TokenProgram](/api-reference/shared-types#tokenprogram).
</ResponseField>
